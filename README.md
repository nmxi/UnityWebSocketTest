# UnityWebSocketTest

## Description
Test to send data from Unity to Node.js server using WebSocket.
Include UniRX and websocket-sharp and use it.

UnityからNode.jsサーバにWebSocketを使って座標情報を送信するテストです．
UniRXとwebsocket-sharpを入れてから動かして下さい．
